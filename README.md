# Dotfiles

In deze repository vind je al mijn dotfiles die ik gebruik tijdens het werken. Ze zijn specifiek aangepast op mijn wensen, maar neem gerust een kijkje als voorbeeld.

### Programma's
1. OS: MacOS (thuis) en Windows (Werk)
2. Editor: <s>Visual Studio Code</s> [emacs](https://www.gnu.org/software/emacs/)
3. Color Theme: Monokai
4. Multiplexer: [Tmux](https://github.com/tmux/tmux/wiki)
5. Terminal:  [Ghostty](https://ghostty.org/) (thuis) en [HyperJS](https://hyper.is/) (werk)
6. Browser: Google Chrome
7. SSH: [1password](https://1password.com)
8. Shell: [Zsh](https://ohmyz.sh/)
